## ✨ 一眼看懂：为什么你需要 Zerodesk？

想象一下：  
敲几个字 → 瞬间打开任意文件/文件夹/网站  
复制内容 → 历史随时呼出，一键粘贴  
常用文本 → 全局热键秒输入  
这一切，**内存只占 5~15MB**，后台安静如无物。

Zerodesk 就是那个让你**再也不想回 Windows 自带搜索**的生产力加速器！

## 核心亮点（全部免费起步，高级功能可选付费）

- 🚀 **闪电搜索与访问**  
  文件、文件夹、网站、应用程序 → 输入即匹配，Enter 直达

  自动记录最近 30+ 条（付费无限），支持文本/图片/文件，快速搜索粘贴

- 🌙 **现代极简界面**  
  支持深色/浅色模式，随系统自动切换，颜值在线

- 🪶 **极致轻量**  
  启动快、常驻低、零广告、无遥测（隐私放心）

- 🔄 **全平台原生体验**  
  Windows / macOS / Linux 统一操作逻辑

## 安装只需 10 秒

1. 前往 **[Releases 页面](http://www.xinghehuimeng.com.cn/zerodeskportal#download)**  
2. 下载适合你系统的安装包：
   - Windows → `Zerodesk-Setup.exe` 或 `.nsis`  
   - macOS → `.dmg`  
   - Linux → `.AppImage` / `.deb` / `.rpm`
3. 双击安装 → 首次运行可能出现 SmartScreen 提示（新软件正常现象），点击「更多信息」→「仍要运行」

**后续下载量增加后，Windows 会自动信任。**
